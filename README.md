# 🧬 Bioinformatics Toolkit

A lightweight Python-based toolkit for basic bioinformatics sequence analysis.  
Designed for students and researchers to perform essential operations on DNA sequences.

---

## 🚀 Features

- 📊 GC content calculation  
- 📂 FASTA file parsing  
- 🔬 DNA → Protein translation  
- 🧬 ORF (Open Reading Frame) detection  
- 💻 Command-line interface (CLI)
